# Installation

```sh
cd $HOME

git clone https://github.com/PasdarTeam/BulletProofCli.git

cd BulletProofCli

chmod +x bullet.sh

./bullet.sh install

./bullet.sh

```
# One command

```sh
cd $HOME && git clone https://github.com/PasdarTeam/BulletProofCli.git && cd BulletProofCli && chmod +x bullet.sh && ./bullet.sh install && ./bullet.sh
```
